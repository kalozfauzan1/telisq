# Documentation Query Protocol

1. Always query Context7 documentation before answering library-related questions
2. Include code snippets from official documentation
3. Specify the exact library version being referenced
4. Note any breaking changes or deprecations

## When to Use This Skill

- User asks about library usage or API
- Question involves specific framework patterns
- Need to verify correct syntax or parameters
- Explaining how to use third-party tools

## Process

1. **Identify the library/framework** in question
2. **Query Context7** for the most recent documentation
3. **Provide specific examples** from the documentation
4. **Include version information** when relevant
5. **Note any caveats** or special considerations
