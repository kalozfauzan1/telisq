# Progress Tracking Protocol

1. Always establish clear milestones
2. Track both completed and pending work
3. Identify blockers and risks
4. Report status clearly and concisely

## When to Use This Skill

- Managing long-term projects
- Tracking multi-step implementations
- Reporting status to stakeholders
- Identifying project bottlenecks

## Process

1. **Define milestones** - Key checkpoints
2. **Track current status** - What's done/remaining
3. **Identify blockers** - Issues preventing progress
4. **Assess timeline** - On track or delayed
5. **Report metrics** - Quantitative progress indicators
6. **Plan next steps** - Immediate priorities
7. **Adjust estimates** - Based on actual progress
