## How to Use

1. Place this command definition in your KiloCode configuration
2. When you run the `init` command, it will:
   - Analyze the current codebase
   - Discover non-obvious project-specific information
   - Create or update AGENTS.md files with only essential information
   - Focus on information that would surprise an experienced developer
   - Preserve existing designed rules/workflows and apply only additive, minimal improvements
   - Keep the files concise and valuable

This command ensures that AGENTS.md files contain only the most important, non-obvious information that AI assistants need to be immediately productive in the codebase.
